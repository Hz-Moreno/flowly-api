# Habit Tracker & Time Tracking API

Este projeto é uma API de Gestão de Hábitos e Time Tracking, criada com o objetivo de aprimorar habilidades de desenvolvimento back-end utilizando o mínimo possível de dependências externas. A proposta é implementar manualmente a maior parte das funcionalidades, seguindo boas práticas de arquitetura e organização de código.

---

## Recursos Principais

- **Gestão de Hábitos:** criação, acompanhamento e marcação de hábitos diários.  
- **Time Tracking:** registro e monitoramento do tempo dedicado a tarefas ou atividades específicas.

---

## Tecnologias Utilizadas

- **Node.js**  
- **TypeScript**  
- **MySQL**  
- **Docker**

---

## Dependências

O projeto utiliza apenas bibliotecas essenciais:

- **mysql2:** driver otimizado para conexão assíncrona com MySQL.  
- **jest:** framework de testes unitários para garantir confiabilidade e cobertura de código.

---

## Arquitetura

O projeto segue os princípios de **Clean Architecture**, garantindo:

- Separação clara de responsabilidades  
- Manutenção facilitada  
- Alta capacidade de adaptação  
- API estruturada em padrão **RESTful**, utilizando corretamente os métodos HTTP (GET, POST, PUT, DELETE)

---

## Testes da API

O repositório inclui uma coleção do **Insomnia** para testar todos os endpoints.
